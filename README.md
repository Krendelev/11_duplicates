# Anti-Duplicator

Simple script that finds all the duplicates (e.g. files with the same name and size) in directory provided.

## Quick start

```bash
$ python duplicates.py test_dir
Duplicates found:
test_dir/1/file2
test_dir/4/file2
test_dir/3/file2

test_dir/1/file1
test_dir/4/file1
```
